### Functionalities

When building a new `Path` object, you need to provide the path to be interpreted.

The available methods are:

- getDirectory
- getExtension
- getFile
- getName
- getPath
- getAbsolutePath
- getInsidePath
- isLocalPath
- isRelativePath
